# DDL: 数据定义语言

## 基础知识

-标识符命名规则
	- 不能超过 30 个字符
	- 只包含 A-Z, a-a, 0-9, _
	- 不能有空格
	- 保持字段名和类型的一致性 

## 创建和管理数据库

```sql
# 默认字符集
CREATE DATABASE mytest1;
# 指明字符集
CREATE DATABASE mytest2 CHARACTER SET 'gbk';
# 只不存在时创建
CREATE DATABASE IF NOT EXISTS mytest2  CHARACTER SET 'utf8'

# 查看数据库
SHOW DATABASES;
# 查看创建过程
SHOE CREATE DATABASE mytest1;
```
